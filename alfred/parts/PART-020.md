### PART-020: CSS Variables & Colors
**Purpose:** CSS custom properties auto-loaded by Components package.
**Key variables:** `--mathai-primary` (#270f36), `--mathai-green` (#219653), `--mathai-red` (#E35757), `--mathai-blue` (#667eea), `--mathai-game-max-width` (480px), `--mathai-game-padding-top` (54px)
**Key rules:**
- Variables load automatically with Components package — no extra `<link>` tag needed
- Use these variables instead of hardcoded colors
- Gameplay cells: `--mathai-cell-*` for default/selected/correct/incorrect/locked states
