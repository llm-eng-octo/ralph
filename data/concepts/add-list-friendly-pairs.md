### Game Concept: Add List with Friendly Pairs — Equivalent Fractions Matching

> **Note:** Upstream worksheet labels this chunk "Add list with friendly pairs", but the captured block content describes an Equivalent Fractions Matching screen. The file below follows the captured content.

This game pairs a numerical fraction with a set of visual area models so the student can match the symbolic fraction to the picture that represents the same portion. It trains part-whole reasoning and the ability to read shaded shape diagrams as fractional quantities.

### Core Mechanics & Interface

* **Status Bar:** A compact header shows a segmented progress bar (10 total segments, 0 filled, yellow) and a trio of red heart lives.
* **Target Fraction:** A large vertical fraction is displayed prominently — a numerator of **1** above a horizontal bar with a denominator of **2** — framing the single instruction "Choose the picture that matches the fraction."
* **Options Area:** A **single row of three large option cards** sitting below the target fraction — evenly spaced, each a white card with a thin gray border. Each card contains a different geometric shape with some regions shaded:
  * A regular hexagon split into 6 equal triangles with 3 shaded yellow (the entire left half shaded, reading as 3/6).
  * A square divided into a 2×2 grid with 3 of the 4 small squares shaded blue (reading as 3/4).
  * A circle split into 4 quadrants with 3 shaded pink (reading as 3/4).
* **Progression:** The game runs across 3 rounds; one correct pick per round is needed to advance.

### Behaviors and Interactions

* **Card Tap:** The student taps the card they believe matches the target fraction. A correct match advances the round; an incorrect match presumably consumes a life from the heart counter shown in the status bar.

### Victory / Completion

Successfully identify the correct visual model for the fraction across all 3 rounds to complete the game.
