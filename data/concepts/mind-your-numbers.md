### Game Concept: Mind Your Numbers — Intersecting Circles Pattern Puzzle

This is a pattern-recognition puzzle built on overlapping circles. The student sees three clusters of four outer numbers surrounding a central intersection value. Two of the clusters are fully labelled and act as worked examples; the third has a "?" in the centre that the student must work out from the pattern the first two establish. It trains inductive reasoning and the ability to test multiple arithmetic relationships.

### Core Mechanics & Interface

* **Instruction Text:** The screen opens with "Figure out the pattern in this image" and "Type the number that should replace the question mark to complete the pattern."
* **Cluster Structure:** Each cluster is a **2×2 composite of overlapping circles** — four outer circles arranged in a square so that their intersection defines a small central circle that holds the answer value. Outer circles carry the four peripheral numbers, and the central overlap holds the derived number. The three clusters are stacked roughly in an L: two worked examples on the upper half of the screen side-by-side, and the target cluster below.
* **Pattern Group 1 (worked example, yellow outlines, blue/purple text):** top-left **2**, top-right **6**, bottom-left **4**, bottom-right **3**, centre **12**.
* **Pattern Group 2 (worked example, purple outlines, magenta text):** top-left **4**, top-right **4**, bottom-left **8**, bottom-right **2**, centre **16**.
* **Pattern Group 3 (target, pink outlines, gold text):** top-left **9**, top-right **2**, bottom-left **6**, bottom-right **3**, centre displayed as a stylised 3D gold **"?"**.
* **Answer Input:** Below the target cluster, a single rectangular input box with a thin gray border and a **"Type here"** placeholder.
* **Progression:** One round; the entire puzzle resolves on a single correct numeric entry.

### Behaviors and Interactions

* **Answer Input (Type):** The student taps the input field to open a keyboard and types the numeric value that replaces the question mark.

### Victory / Completion

Typing the correct number that satisfies the pattern across all three clusters completes the puzzle.

### Variant Progression

The three variants describe the same three-circle-cluster pattern puzzle with the same outer numbers and the same centre values (12, 16, ?) — the cluster outline/text colours shift slightly between B1 and B2, and B3 wraps the instruction in a "▯ Figure out the pattern ▯" decoration but keeps the mechanic and the single typed-answer interaction identical.
